My Amazing Works!!!

Complaint Segregation Application

Developed a powerful sentiment analysis program for Twitter data, streamlining the identification and classification of customer complaints to enhance business service and reputation. The solution automated tweet extraction using the SOAP protocol, integrated MBART to translate non-English tweets into English for accurate sentiment analysis, and employed the RoBERTa model to assess sentiment polarity and intensity. By leveraging sentiment scores to prioritize complaints, the system reduced manual testing by 80% and enabled targeted responses. It also featured visualization tools to track complaint trends and resolution effectiveness.

Uber Data Analytics using GCP Services

This project leverages the "NYC 2023" dataset and Google Cloud Platform (GCP) to enhance public transportation accessibility in urban areas. By analyzing data and collaborating with stakeholders, we aim to improve transit quality for all residents. The project involves data acquisition, processing with Mage ETL, storage in Google Cloud Storage, analysis in BigQuery, and visualization in Looker dashboards. Prioritizing data security and scalability, we measure success through increased ridership, accessibility, reduced congestion, and rider satisfaction. This project serves as a model for using GCP in urban transit solutions.
